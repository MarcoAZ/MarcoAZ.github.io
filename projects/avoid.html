Avoid

You are a cowardly, yellow rectangle and do not want to be touched by those circles! This project came about when I felt I was ready to create my own game from start to finish. To come up with a simple game, I thought back to some of the games I’d play when we’d have access to computers during class. I remembered a game close to what you see here keeping us occupied for long periods of time.

Avoid is when I truly began to really build code on my own and watch it all come together… or come crashing down. I made several mistakes but I’m very proud of this end product. Though I was still wary of working with real art assets, I challenged myself to create some interesting effects with the HTML canvas to keep things interesting.
